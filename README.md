# Follow Us Everywhere Prototype

A clickable React prototype for a business-only follow hub that lets customers connect with a business across social platforms from a single link.

## Quick Start

```bash
npm install
npm run dev
```

Then open the local URL shown in the terminal.

## Available Scripts

- `npm run dev` — start the Vite development server
- `npm run build` — build the production bundle
- `npm run preview` — preview the production build locally
